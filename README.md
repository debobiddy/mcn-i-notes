# mcn-i-notes

## 项目简介

`mcn-i-notes` 是一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为一系列自包含的网页文件的存储与展示空间。

这些页面可能包含笔记、工具、演示或实验性内容，每个页面均为独立的 HTML 文件，可直接在浏览器中打开查看。

## 目录说明

```
mcn-i-notes/
├── pages/          # 归档的 HTML 页面存放目录
├── assets/         # 页面引用的静态资源（如图片、样式、脚本等）
├── index.html      # 仓库主页（可选）
└── README.md       # 本说明文件
```

- `pages/`：每个独立的 HTML 文件放置于此目录下，文件名以简洁描述命名。
- `assets/`：若页面需要额外资源，统一存放于此，便于管理。
- `index.html`：可选的入口页面，用于列出所有归档页面或提供导航。

## 页面归档说明

所有归档的 HTML 页面均为独立文件，不依赖外部网络资源（除正常 CDN 或字体库外）。页面内容可能涵盖技术笔记、文档片段、互动演示等。

每个页面通常包含：
- 完整的 HTML 结构
- 自包含的 CSS 和 JavaScript（或通过 `assets/` 引用）
- 简洁的页面标题与说明

若需新增页面，请将 HTML 文件放入 `pages/` 目录，并确保其可独立运行。
